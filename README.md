# Java OOP - Customer Model

Simple Java console project demonstrating Object-Oriented Programming concepts.

## Concepts used
- Encapsulation
- Constructor overloading
- Method usage (getters)
- Object creation

## Description

This project defines a `Customer` class with:
- name
- credit limit
- email

It demonstrates multiple constructors and basic object usage in a main class.

## How to run

Open the project in IntelliJ IDEA and run `Main.java`.
